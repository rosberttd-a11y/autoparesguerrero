# Autopartes Guerrero

Tienda online de repuestos automotrices.

## Deploy en Vercel

1. Sube este proyecto a un repositorio en GitHub
2. Ve a [vercel.com](https://vercel.com) y conecta tu cuenta de GitHub
3. Importa el repositorio
4. Vercel detectará Next.js automáticamente — click en Deploy

## Desarrollo local

```bash
npm install
npm run dev
```

Abre http://localhost:3000
